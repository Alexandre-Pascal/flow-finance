# Internationalisation (i18n)

## Stack

- [next-intl](https://next-intl.dev/) v4+
- Fichiers de messages : [`messages/fr.json`](../messages/fr.json), [`messages/en.json`](../messages/en.json)

## URLs

| Locale | Exemple |
|--------|---------|
| Français (défaut) | `/fr`, `/fr/accounts` |
| Anglais | `/en`, `/en/settings` |

Le middleware redirige `/` vers `/fr`.

## Ajouter une traduction

1. Ajouter la clé dans `messages/fr.json` et `messages/en.json`
2. Utiliser dans un composant :

```tsx
// Server Component
const t = await getTranslations("dashboard");
t("title");

// Client Component
const t = useTranslations("dashboard");
t("title");
```

## Changer de langue

Composant [`LanguageSwitcher`](../src/components/features/language-switcher.tsx) dans Paramètres — utilise `router.replace(pathname, { locale })`.

## Métadonnées

`generateMetadata` dans [`src/app/[locale]/layout.tsx`](../src/app/[locale]/layout.tsx) lit `metadata.title` et `metadata.description` depuis les messages.
